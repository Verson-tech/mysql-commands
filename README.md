- To list available databases:

  ```
  show databases;
  ```

- The general command for creating a database:

  ```
  CREATE DATABASE <database_name>;
  ```

- To drop a database:

  ```
  ROP DATABASE <database-name>;
  ```

- To use a database:

  ```
  USE <database-name>;
  ```

- Create a table:

```
CREATE TABLE cats (
    name VARCHAR(50),
    age INT
);

CREATE TABLE dogs (
    name VARCHAR(50),
    breed VARCHAR(50),
    age INT
);
```
